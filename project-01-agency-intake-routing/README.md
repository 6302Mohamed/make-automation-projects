# Project 01: Agency Intake Routing

## Overview

This project is a Make.com workflow for handling agency request submissions from a Google Form connected to Google Sheets.

It demonstrates how to validate incoming requests, detect duplicates, route submissions based on business rules, and update records for audit visibility.

## Tools used

- Make.com
- Google Forms
- Google Sheets
- Email modules

## Main features

- Required-field validation
- Invalid request handling
- Duplicate detection
- Priority-based routing
- Category-based email routing
- Status updates for audit visibility
